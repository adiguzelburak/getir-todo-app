# Welcome to Getir-Todo-App

## Available Scripts

Run `npm install` or `yarn` to install dependencies

Run `npm start` or `yarn start` to start Web App

- React JS
- Redux
- Redux Saga
- Redux Logger
- Firebase (Database)

## UI Libraries

- Bootstrap
- Lottie
- Sweetalerts
